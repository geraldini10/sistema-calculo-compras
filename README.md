# 🛒 Sistema de Cálculo de Compras em Java

Sistema de cálculo de compras com diversas regras de negócio implementadas do zero!

## O programa calcula automaticamente:

- Desconto por faixa de valor (5% a 22%)
- Descontos adicionais por tipo de cliente (Comum, VIP e Premium)
- Descontos por forma de pagamento (à vista, débito e crédito)
- Juros de parcelamento (até 12x)
- Frete automático baseado no valor da compra
- Cupons promocionais com validações (BEMVINDO10, SUPER15 e FRETE0)
- Bônus de aniversário e promoções sazonais

## Tecnologias
- Java
- Lógica de programação
- Estruturas condicionais
